重要提醒
  • JSON 檔案必須包含 contest_id，系統會自動讀取
  • 資料庫必須事先有建立好相同 ID 的競賽、題目和使用者
  • 若提交記錄的 ID 已存在，則會直接覆蓋該筆資料
  • 匯入過程中若遇到錯誤（例如競賽、使用者或題目不存在），該筆資料會被跳過
  • 只適用於同一系統內的資料移轉，跨系統請使用「完整競賽匯入」
  • 建議先在測試環境測試匯入功能
匯入前檢查清單
  • 目標競賽已建立
  • 所有題目已存在
  • 所有使用者帳號已建立
  • JSON 檔案格式正確
  • 已備份現有資料
使用場景
  • 資料庫遷移
  • 複製競賽資料到另一系統
  • 資料備份還原
  • 測試環境資料準備
